# AI 日报目录

此目录存储每日自动生成的 AI 资讯日报。

## 文件命名规则

- 格式：`YYYY-MM-DD.md`
- 示例：`2026-01-20.md`

## 内容分类

每份日报包含以下内容：

1. 🚀 重大发布 - 新模型、新产品
2. 🤖 Agent 技术进展 - 自主 Agent、多 Agent 系统
3. 📊 最新论文 - 顶会论文、arXiv 热门
4. 🛠️ 开源项目 - 新工具、框架更新
5. 💡 技术突破 - 算法创新、性能突破
6. 💼 行业动态 - 公司新闻、融资、政策

## 生成方式

- **自动生成**：每天早上 9:00 由 Celery Beat 定时任务触发
- **手动生成**：执行 `claude -p "执行 /ai-digest"`

## 查看最新日报

```bash
# 查看今日日报
cat main/docs/ai_digest/$(date +%Y-%m-%d).md

# 查看最近 7 天日报
ls -lt main/docs/ai_digest/ | head -8
```
