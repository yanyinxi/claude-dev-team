# Examples - 学习示例库

## 4 个示例库

| 示例 | 难度 | 内容 |
|------|------|------|
| [Simple CRUD](simple_crud/) | ⭐ | 基础增删改查操作 |
| [Bug Fix](bug_fix/) | ⭐⭐ | 调试真实缺陷 |
| [Todo App](todo_app/) | ⭐⭐⭐ | 完整全栈应用 |
| [Fullstack App](fullstack_app/) | ⭐⭐⭐⭐ | 企业级电商系统 |

## 快速开始

```bash
cd todo_app && ./start.sh    # 推荐：完整全栈应用
cd simple_crud              # 新手：只学增删改查
cd bug_fix                  # 学习如何调试缺陷
cd fullstack_app            # 企业级项目案例
```

每个文件夹都有 README，点击上面链接查看详情。

## 🎯 Todo App - 全栈应用示例 ⭐ 推荐

**难度**：⭐⭐⭐ 中级 | **时间**：1-2 小时 | **推荐对象**：全栈开发者

**位置**：`todo_app/`

一个完整的待办事项管理应用，展示从需求分析到上线部署的整个开发流程。

**快速开始**：
```bash
cd todo_app && ./start.sh
```

然后访问 http://localhost:3000

**包含内容**：
- ✅ React + TypeScript 前端
- ✅ Express.js + SQLite 后端
- ✅ 完整的单元测试
- ✅ **详细的操作日志** ← 实时监控 CRUD
- ✅ 一键启动脚本
- ✅ API 文档和产品文档

**[查看详情 →](todo_app/README.md)**

## 🛒 Fullstack App - 电商系统示例

**难度**：⭐⭐⭐⭐ 高级 | **时间**：1-2 周 | **推荐对象**：高级开发者/架构师

**位置**：`fullstack_app/`

展示一个完整的电商系统开发案例，涵盖用户、商品、订单、支付、物流等完整功能。

**快速开始**：
```bash
cd fullstack_app && cat README.md
```

**涵盖内容**：
- 产品需求文档
- 系统架构设计
- 数据库设计
- 完整的代码实现
- Docker 部署
- 运维和监控

**[查看详情 →](fullstack_app/README.md)**
