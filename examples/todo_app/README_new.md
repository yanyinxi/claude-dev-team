# Todo App - 完整全栈应用

**难度**：⭐⭐⭐ | **时间**：1-2 小时 | **推荐入门**

## 背景

完整的待办事项应用。展示从产品需求到上线运行的整个开发过程。包含 React 前端、Express 后端、SQLite 数据库、测试、日志等。

## 文件结构

```
├── src/
│   ├── backend/app.js         # Express 服务器 + 路由
│   ├── frontend/index.html    # React 入口
│   └── data/todos.db          # 数据库
├── tests/                     # 自动化测试
├── package.json
├── start.sh                   # 一键启动脚本
└── README.md                 # 本文件
```

## 如何实现

```bash
./start.sh                     # 自动：检查环境 -> 安装依赖 -> 初始化DB -> 启动服务
```

访问 http://localhost:3000 查看应用 UI。
