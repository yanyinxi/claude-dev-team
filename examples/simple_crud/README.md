# Simple CRUD - 基础增删改查

**难度**：⭐ | **时间**：30 分钟

## 背景

学习最基本的数据操作：Create（创建）、Read（读取）、Update（更新）、Delete（删除）。

## 文件结构

```
├── input/requirement.txt      # 功能需求
├── expected/                  # 参考实现
├── walkthrough.md            # 开发步骤
└── README.md                 # 本文件
```

## 如何实现

1. `cat input/requirement.txt` - 了解需求
2. `cat walkthrough.md` - 学习步骤
3. `cat expected/` - 查看参考答案

## 如何使用

跟着 `walkthrough.md` 逐步实现 4 个操作，理解每一步的原理。

## 注意事项

- 先理解概念，再看代码
- 修改代码后运行测试验证
- 尝试添加新字段，加深理解
