## 📚 可用示例

### 1. 🎯 Todo App - 全栈应用示例 ⭐ 推荐

**位置**: `todo_app/`

一个完整的待办事项管理应用，展示从需求分析到上线部署的整个开发流程。

**包含内容**:
- ✅ React + Express 全栈应用
- ✅ SQLite 数据库
- ✅ 完整的单元测试
- ✅ API 文档
- ✅ 产品需求文档
- ✅ 技术设计文档
- ✅ 开发教程

**快速开始**:
```bash
cd todo_app
npm install
npm run init-db
npm start
```

然后访问 http://localhost:3000

**学习时间**: 30-60 分钟  
**难度级别**: 中级  
**最适合**: 学习全栈开发流程

**推荐阅读**:
1. [快速入门](todo_app/QUICKSTART.md) - 5 分钟快速体验
2. [详细文档](todo_app/README.md) - 完整使用指南
3. [逐步教程](todo_app/WALKTHROUGH.md) - 深入学习路径
